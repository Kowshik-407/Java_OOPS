 JDK vs JRE
 
 JDK --> Java Development Kit
		- Compiler, libraries, Runtime, setup, JVM
		- Developers
 JRE --> Java Runtime Environment
		- Runtime, setup, JVM
		- End User

 JVM --> Java Virtual Machine
 
 Code(source.java) -> Compile Code -> Byte Code (Optimized code for machine may be in assembly language) -> JVM
